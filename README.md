# Solar Powered E-Paper Display for Digital Advertising

This project demonstrates a **sustainable digital advertising system** powered by **solar energy** and displayed on an **E-Paper screen**.  
E-Paper consumes extremely low power and works excellently in sunlight, making it ideal for green smart-city applications.
